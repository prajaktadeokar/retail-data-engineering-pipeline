# retail-data-engineering-pipeline
This project builds an end-to-end data pipeline using Python, PySpark, Airflow, and Azure.
## Architecture

Source → Data Lake → Spark → Warehouse → Dashboard

## Tech Stack

Python
PySpark
Airflow
Azure Data Lake
Azure Databricks
Azure Synapse
Power BI

## Pipeline Steps

1 Data ingestion from CSV
2 Data stored in Azure Data Lake
3 PySpark transformations
4 Airflow DAG orchestration
5 Load into warehouse
6 BI dashboard
